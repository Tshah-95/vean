# vean sidecars

The production Mac app may bundle pinned subprocess sidecars here:

- `vean`
- `vean-lsp`
- `vean-mcp`
- `melt`
- MLT modules and profiles
- `ffmpeg`
- `ffprobe`

This directory is intentionally a manifest placeholder in the seed app. The
source/CLI/Homebrew distribution continues to use system `mlt` and `ffmpeg`
dependencies.
